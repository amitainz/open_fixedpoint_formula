Smaller Modules
===============


These are smaller modules that have quite specific functionality.

jrrFormula module
-----------------

This module implements the Pandharipande-Solomon-Tessler formula for open descendent integrals (of a point). See Section 1.7 of `their paper <https://arxiv.org/abs/1409.2191>`_.

dlk_partitions module
---------------------

This generates all partitions of a given total degree and number of interior and boundary markings. Each element of the partition is subject to a stability, orientability, and sign-of-conjugation condition (:py:func:`goodDLK_2` checks for this)
